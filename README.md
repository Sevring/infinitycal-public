# infinitycal-public

Public-facing notices for the **무한 사칙연산 (Infinity Calc)** Android app.
This repository is intentionally separated from the (private) app source
repository so that the Google Play store listing can link to a stable
public URL without exposing the application code.

## 페이지 목록

| 경로 | URL | 설명 |
| --- | --- | --- |
| `privacy/` | https://sevring.github.io/infinitycal-public/privacy/ | 개인정보처리방침 |

향후 추가될 페이지 (예정)

- `terms/` — 이용약관
- `notice/` — 공지/패치노트
- `support/` — 문의/지원
