무한 사칙연산 (Infinity Calc) 개인정보처리방침

개정일자: 2026년 5월 27일 · 운영자: areyoupasta
요약 — 본 앱은 광고, 추적, 분석 SDK를 포함하지 않으며, 어떠한 사용자 데이터도 외부로 전송하지 않습니다. 모든 학습 통계와 설정값은 사용자의 단말기 내부 저장소에만 보관됩니다.

1. 수집하는 정보

본 앱은 다음 정보를 단말기 내부에만 저장합니다.

  • 학습 통계: 누적 문제 수, 정답 수, 정답률, 현재·최고 연속 정답 기록, 누적 풀이 시간.
  • 앱 설정값: 출제 연산자, 자릿수 범위, 나눗셈 모드, 소수점 자릿수, 자동 진행 여부, 햅틱/사운드 여부, 테마 모드 등.

위 정보는 Android의 SharedPreferences 영역에 저장되며, 본 앱 외부의 어떤 서버나 제3자에게도 전송되지 않습니다.

2. 수집하지 않는 정보

  • 이름, 이메일, 전화번호, 주소 등 개인 식별 정보
  • 위치 정보(GPS), 연락처, 카메라, 마이크, 사진 등 단말기 자원
  • 광고 식별자(Advertising ID), 단말기 식별자(IMEI, MAC 등)
  • 크래시 로그, 사용 분석 데이터, 광고 클릭 데이터

3. 사용 권한

본 앱은 어떠한 Android 권한도 요청하지 않습니다. 네트워크 통신을 위한 INTERNET 권한 또한 선언되어 있지 않습니다.

4. 제3자 정보 제공 및 SDK

본 앱은 어떠한 광고, 추적, 분석 SDK도 포함하지 않습니다. 따라서 제3자에게 제공되는 정보가 존재하지 않습니다.

5. 사용자 데이터에 대한 권리

본 앱에 저장된 모든 데이터는 사용자가 직접 삭제할 수 있습니다.

  • 앱 내 설정에서 통계 초기화를 실행하면 누적 통계가 즉시 0으로 재설정됩니다.
  • Android 설정 → 애플리케이션 → 무한 사칙연산 (Infinity Calc) → 저장공간 → 데이터 삭제를 실행하면 설정값과 통계가 모두 영구 삭제됩니다.
  • 앱을 삭제하면 단말기에 저장된 모든 데이터도 함께 삭제됩니다.

6. 아동 보호

본 앱은 만 13세 미만 아동을 주된 대상으로 하지 않으나, 광고와 외부 데이터 송신이 없으므로 모든 연령에서 안전하게 이용할 수 있습니다.

7. 정책 변경

본 방침이 변경될 경우 이 페이지의 개정일자가 갱신됩니다. 중요한 변경 사항이 있을 경우 앱 업데이트 노트에 함께 안내합니다.
