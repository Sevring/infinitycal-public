무한 사칙연산 (Infinity Calc) 개인정보처리방침

개정일자: 2026년 5월 27일 (v2) · 운영자: areyoupasta
요약 — 본 앱은 무료 사용을 위해 Google AdMob 배너 광고를 표시합니다. 학습 통계와 설정값은 사용자 단말기 내부에만 저장되며 외부로 전송되지 않습니다. 본 앱은 사용자 추적이나 분석을 위한 별도 SDK 를 사용하지 않으며, AdMob 이 광고 표시 과정에서 수집하는 정보는 아래에 자세히 안내합니다.

1. 본 앱이 단말기에 저장하는 정보

  • 학습 통계: 누적 문제 수, 정답 수, 정답률, 현재·최고 연속 정답 기록, 누적 풀이 시간.
  • 앱 설정값: 출제 연산자, 자릿수 범위, 나눗셈 모드, 소수점 자릿수, 자동 진행 여부, 햅틱/사운드 여부, 테마 모드 등.

위 정보는 Android 의 SharedPreferences 영역에 저장되며, 본 앱의 외부 서버나 제3자에게 전송되지 않습니다.

2. 광고를 통해 수집/처리되는 정보 (Google AdMob)

본 앱은 Google AdMob 의 배너 광고를 메인 화면 하단에 표시합니다. AdMob 은 광고를 게재하기 위해 다음 정보를 자동으로 처리할 수 있습니다.

  • Android 광고 ID (Advertising ID)
  • IP 주소 및 대략적인 위치 정보 (국가/지역 수준)
  • 단말기 모델·OS 버전·언어·시간대
  • 광고 노출 및 클릭 이벤트

위 정보는 Google 의 개인정보처리방침에 따라 처리되며, 자세한 내용은 아래 링크를 참고해 주세요.

  • Google 개인정보처리방침
  • AdMob 이 수집·사용하는 데이터

3. 본 앱이 수집하지 않는 정보

  • 이름, 이메일, 전화번호, 주소 등 본 앱이 직접 묻는 개인 식별 정보
  • 정확한 위치 정보(GPS), 연락처, 카메라, 마이크, 사진 등 단말기 자원
  • 크래시 로그, 사용 분석 데이터 (별도 분석 SDK 미사용)

4. 사용 권한

  • INTERNET — 광고를 불러오기 위해 사용. 본 앱은 학습 데이터를 인터넷으로 전송하지 않습니다.
  • com.google.android.gms.permission.AD_ID — AdMob 이 광고 ID 를 읽기 위해 사용. Android 설정에서 사용자가 언제든 재설정/제한 가능합니다.

5. 제3자 SDK

  • Google AdMob (google_mobile_ads) — 광고 게재.

위 외의 추적/분석/광고 네트워크 SDK 는 포함되어 있지 않습니다.

6. 사용자 데이터에 대한 권리

  • 앱 내 설정에서 통계 초기화를 실행하면 단말기에 저장된 누적 통계가 즉시 0 으로 재설정됩니다.
  • Android 설정 → 애플리케이션 → 무한 사칙연산 (Infinity Calc) → 저장공간 → 데이터 삭제를 실행하면 설정값과 통계가 모두 영구 삭제됩니다.
  • Android 설정 → 개인정보 → 광고 → 광고 ID 재설정 또는 광고 ID 삭제 를 실행하면 광고 개인 맞춤이 리셋/비활성화됩니다.
  • 앱을 삭제하면 단말기에 저장된 본 앱 관련 데이터가 모두 삭제됩니다. AdMob 이 보관하는 광고 식별자/이력은 Google 의 개인정보처리방침에 따라 처리됩니다.

7. 아동 보호

본 앱은 만 13세 미만 아동을 주된 대상으로 하지 않습니다. AdMob 광고는 Google Play 의 콘텐츠 등급(전체 이용가) 에 맞추어 가족 친화적 (Family-friendly) 광고만 게재되도록 설정되어 있습니다.

8. 정책 변경

본 방침이 변경될 경우 이 페이지의 개정일자가 갱신됩니다. 중요한 변경 사항은 앱 업데이트 노트에 함께 안내합니다.
